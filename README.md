## 功能说明
### 坐标记录
#### 命令
`/sc pos list` 列出所有记录的坐标并给予玩家一本相同内容的书

`/sc pos here <地名> <简介>` 添加一条坐标记录

`/sc pos search <关键词>` 查询坐标记录

### 生存辅助
#### 消除灾厄
拿着剑右键消除灾厄并播放音效
#### 经验修补增强
玩家潜行状态下右键带有经验修补附魔的工具时会用自己的经验修复工具

### 模组联动
* 坐标记录支持与XWM、XMM和OMMC联动
    * 支持高亮显示和记录坐标点

### 小彩蛋
* 攻击保护动物坤坤会被惩罚
